# 法律平台首頁開發筆記

## 設計實現

### 色彩系統
- **主色**：深藍色 (#0066CC) - 用於主要 CTA 按鈕和強調
- **輔助色**：翠綠色 (#00D084) - 用於次要強調、邊框和標籤
- **背景**：深灰色 (#1A1A1A) - 主背景色
- **文字**：白色 (#FFFFFF) - 主文字色
- **卡片背景**：#2D3748 - 功能卡片背景
- **邊框色**：#4A5568 - 邊框和分隔線

### 排版系統
- **標題字體**：Poppins (700 Bold) - 用於所有標題
- **正文字體**：Inter (400 Regular) - 用於描述和正文
- **強調文字**：Inter (600 SemiBold) - 用於按鈕和標籤

### 視覺資產
1. **Hero 背景影像** (`/images/hero-background.jpg`)
   - 深藍色和深灰色漸變
   - 專業企業級風格
   - 配合 60% 透明度覆蓋層確保文字可讀性

2. **幾何紋圖案** (`/images/geometric-pattern.png`)
   - 灰銀色六邊形圖案
   - 用於 Hero Section 右上角裝飾
   - 增加設計細節和視覺層次

### 頁面結構

#### 1. Hero Section
- 全屏背景影像 + 覆蓋層
- 居中文本內容
- 雙按鈕 CTA 設計（主按鈕 + 次按鈕）
- 滾動指示器（向下箭頭動畫）

#### 2. 核心功能區
- 6 個功能卡片（3x2 網格）
- 翠綠色圖標
- 懸停效果：邊框變綠、陰影加深
- 響應式設計

#### 3. 高級功能區
- 3 個高級功能卡片
- 深色背景 + 綠色邊框
- 企業版標籤
- 懸停效果

#### 4. 價格表區
- 4 個價格卡片
- Professional 卡片突出顯示（綠色邊框）
- 推薦標籤
- 清晰的功能列表

#### 5. 頁尾
- 簡潔設計
- 公司名稱和版權信息

### 互動效果
- **按鈕懸停**：顏色加深、陰影效果
- **卡片懸停**：邊框變綠、陰影加深、背景微調
- **圖標懸停**：縮放效果 (scale-110)
- **滾動指示器**：連續向下動畫

### 響應式設計
- 移動優先設計
- 平板和桌面斷點
- 功能卡片自適應網格
- 按鈕堆疊在小螢幕上

## 技術實現

### 框架和工具
- **React 19** - 前端框架
- **Tailwind CSS 4** - 樣式系統
- **shadcn/ui** - UI 組件庫
- **Wouter** - 路由管理

### 關鍵文件
- `client/src/pages/Home.tsx` - 首頁主組件
- `client/src/index.css` - 全局樣式和色彩系統
- `client/index.html` - HTML 模板（包含 Google Fonts）
- `client/public/images/` - 視覺資產目錄

### 自定義 CSS 變數
所有顏色和主題設置在 `index.css` 中定義，支持深色/淺色主題切換。

## 對標 Clio 官網的設計元素

1. **深色背景 + 背景影像**：營造專業、現代的企業形象
2. **高對比度文字**：白色文字在深色背景上確保可讀性
3. **三色系統**：深藍 + 綠色 + 白色建立清晰的視覺層級
4. **精準的排版**：Poppins + Inter 字體組合
5. **微妙的動畫**：懸停效果和過渡提升互動感
6. **幾何裝飾**：灰銀色圖案增加設計細節
