# 法律平台首頁設計理念

## 選定設計方案：專業企業級深色系設計

### 設計哲學
參考 Clio 官網的現代企業級設計語言，打造一個專業、信任感強、視覺衝擊力大的法律服務平台首頁。設計強調**深色背景 + 動態影片 + 高對比度文字 + 精準的色彩層級**，營造專業感與現代感並存的氛圍。

### 核心設計原則

1. **深色優雅**：採用深灰/黑色漸變背景，搭配動態影片層，營造沉穩、專業的企業形象
2. **高對比度**：白色文字與深色背景形成強烈對比，確保可讀性與視覺衝擊力
3. **色彩精準**：三色系統（深藍、翠綠、白色）建立清晰的視覺層級，引導用戶注意力
4. **動態感**：影片背景 + 微妙的動畫效果，提升整體的現代感與活力

### 色彩哲學

| 色彩 | 用途 | 情感意義 |
|------|------|--------|
| 深藍色 (#0066CC) | 主要 CTA 按鈕、重點強調 | 信任、專業、穩定 |
| 翠綠色 (#00D084) | 次要強調、邊框、標籤 | 成長、活力、創新 |
| 白色 (#FFFFFF) | 主文字、背景對比 | 清晰、簡潔、高級 |
| 深灰色 (#1A1A1A) | 主背景 | 專業、沉穩、現代 |
| 灰銀色 (#4A5568) | 幾何紋圖案、次要文字 | 細節、層次、精緻 |

**情感意圖**：色彩組合傳達「專業可靠的法律服務」與「現代創新的技術平台」的雙重身份。

### 版面編排範式

**非傳統中心對齊**：採用**半居中 + 左側視覺重點**的非對稱佈局
- Hero Section：文字內容略左，背景影片佔據全幅，營造沉浸感
- 功能區塊：採用交替的左右排列，避免單調的網格結構
- 價格表：採用卡片式設計，但加入灰銀色幾何邊框作為裝飾

### 標誌性視覺元素

1. **動態影片背景**：Hero Section 使用循環播放的深色調影片或漸變動畫，營造專業感
2. **灰銀色幾何紋圖案**：在功能卡片、邊框、分隔線中融入幾何圖案，增加設計細節
3. **綠色強調線**：在重點區塊（標題下方、邊框）使用翠綠色線條作為視覺分隔

### 互動設計哲學

- **按鈕設計**：雙層視覺層級（主按鈕：深藍 + 白字；次按鈕：邊框 + 綠色邊框）
- **懸停效果**：按鈕懸停時顏色加深或添加發光效果，提升互動感
- **平滑過渡**：所有過渡使用 300-400ms 的 ease-out 動畫，確保流暢感
- **卡片效果**：功能卡片懸停時添加陰影加深效果，營造浮起感

### 動畫指南

- **入場動畫**：頁面加載時，文字從下方淡入並向上滑動（300ms）
- **懸停動畫**：卡片懸停時，陰影加深、背景色微調（200ms）
- **按鈕反饋**：按鈕點擊時添加脈衝效果或顏色閃爍（150ms）
- **滾動動畫**：功能區塊在滾動進入視口時逐個淡入（500ms）

### 排版系統

| 層級 | 字體 | 大小 | 粗細 | 用途 |
|------|------|------|------|------|
| 主標題 | Poppins | 56-64px | 700 (Bold) | Hero 標題 |
| 副標題 | Poppins | 32-40px | 600 (SemiBold) | 區塊標題 |
| 正文 | Inter | 16-18px | 400 (Regular) | 描述文字 |
| 強調文字 | Inter | 14-16px | 600 (SemiBold) | 按鈕、標籤 |
| 小文字 | Inter | 12-14px | 400 (Regular) | 說明、備註 |

**字體配對**：Poppins（顯示層級，現代感強）+ Inter（正文層級，易讀性高）

### 實作重點

1. 使用 HTML5 `<video>` 標籤實現影片背景，支持自動播放、靜音、循環
2. 添加半透明覆蓋層（深灰色 + 60% 透明度）以確保文字可讀性
3. 在功能卡片中融入灰銀色幾何邊框或背景圖案
4. 使用 Tailwind CSS 的自定義顏色變數管理色彩系統
5. 實現響應式設計，確保在不同螢幕尺寸上的視覺效果一致
