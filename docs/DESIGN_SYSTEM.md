# HK Legal Case Agency - Design System Documentation

## 設計理念

本設計系統遵循「高端穩重」的設計策略，為香港法律市場營造專業、可信的視覺環境。

## 五大設計核心

| 原則 | 說明 | 實踐方式 |
|------|------|---------|
| **專業權威性** | 傳達法律行業的可信度與權威 | 配色保守、排版精緻、對比清晰 |
| **簡潔高效** | 信息架構清晰，避免過度復雜 | 白空間充足、頁面層級明確 |
| **香港本地化** | 符合香港客群的審美習慣 | 配色選擇、文化敏感性設計 |
| **高端穩重感** | 避免鮮豔休閒，體現成熟感 | 使用深色調與經典配色 |
| **移動優先** | 考慮移動端使用場景 | 響應式設計、觸控友好 |

### 色彩應用指南

#### 主要應用場景

**頁面導航區**  
- 背景：深藍綠 (#004B5C)
- 文字：白色 (#FFFFFF)
- 目的：醒目但不突兀，傳達專業感

**主要CTA按鈕**  
- 背景：清新綠 (#3BA19D)
- 文字：白色 (#FFFFFF)
- 用於：「免費試用」、「立即開始」、「預約演示」等高優先級操作

**高級功能強調**  
- 背景：文雅金 (#D4AF37)
- 用於：VIP標籤、企業版獨享功能、白金客戶標識
- 目的：增加香港市場的高級感

**頁面背景與分層**  
- 主背景：乳白色 (#FAFAFA)
- 卡片背景：白色 (#FFFFFF) 或 乳白色 (#FAFAFA)
- 邊框/分隔：淺灰色 (#E8E8E8)
- 目的：避免過度白色冷硬感，層次清晰

**文字層級**  
- 標題：深藍綠 (#004B5C)
- 主文本：石墨黑 (#1A1A1A)
- 副文本：冷灰色 (#5A5A5A)
- 輔助信息：淺灰色 (#E8E8E8)

### 色彩可訪問性標準

所有色彩組合必須符合 **WCAG AA 標準**（對比度 ≥ 4.5:1）：

| 顏色組合 | 對比度 | 符合性 | 使用場景 |
|---------|------|------|---------|
| 深藍綠 + 白色 | 8.2:1 | ✓ | 導航欄、頁腳 |
| 清新綠 + 白色 | 5.1:1 | ✓ | CTA按鈕 |
| 石墨黑 + 乳白色 | 14.2:1 | ✓ | 主體文本 |
| 文雅金 + 石墨黑 | 6.4:1 | ✓ | 企業版標籤 |
| 冷灰色 + 白色 | 4.6:1 | ✓ | 副文本（使用白色背景時） |

## 組件應用示例

### 首頁設計

首頁（`/app/page.tsx`）完整實現了設計系統規範：

1. **Hero 區域**
   - 使用深藍綠背景 (`bg-teal-dark`)
   - 白色主標題，清新綠副標題 (`text-mint-green`)
   - 清新綠主CTA按鈕，白色次要按鈕

2. **核心功能區**
   - 白色背景 (`bg-white`)
   - 清新綠圖標背景 (`bg-mint-green`)
   - 深藍綠標題 (`text-teal-dark`)
   - 冷灰色說明文字 (`text-cool-gray`)

3. **高級功能區（企業版專享）**
   - 乳白色背景，文雅金頂部邊框 (`border-t-4 border-subtle-gold`)
   - 文雅金徽章突出「企業版專享」
   - 文雅金圖標背景和邊框
   - 文雅金標籤標識「企業版獨享」

4. **訂閱計劃區**
   - 白色背景 (`bg-white`)
   - Starter 和 Custom 使用淺灰色邊框
   - Professional 使用清新綠邊框和徽章（推薦）
   - Enterprise 使用文雅金邊框和徽章（企業版）

5. **頁腳**
   - 深藍綠背景 (`bg-teal-dark`)
   - 白色文字

### 按鈕組件

按鈕組件（`/components/ui/button.tsx`）實現了四種變體：

- **Primary**: 清新綠背景，白色文字
- **Secondary**: 深藍綠邊框和文字，懸停時深藍綠背景
- **Text**: 深藍綠文字，懸停時清新綠並帶下劃線
- **Danger**: 警告紅背景，白色文字

## 技術實現

### Tailwind 配置

所有顏色在 `tailwind.config.ts` 中定義：

```typescript
colors: {
  "teal-dark": "#004B5C",
  "mint-green": "#3BA19D",
  "subtle-gold": "#D4AF37",
  "charcoal": "#1A1A1A",
  "cool-gray": "#5A5A5A",
  "light-gray": "#E8E8E8",
  "off-white": "#FAFAFA",
  "info-blue": "#2E6FA8",
  "alert-red": "#D32F2F",
  "success-green": "#388E3C"
}
```

### 全局樣式

`/app/globals.css` 設置基礎樣式：

```css
body {
  font-family: "Inter", system-ui, sans-serif;
}
```

## 設計原則檢查清單

在實現新功能時，請確保：

- [ ] 使用定義的色板，不引入新顏色
- [ ] 文字與背景對比度符合 WCAG AA 標準（≥ 4.5:1）
- [ ] 使用文雅金突出高級/企業版功能
- [ ] 主要操作使用清新綠按鈕
- [ ] 保持充足的白空間和清晰的層級
- [ ] 響應式設計支持移動端
- [ ] 保持專業、穩重的視覺風格
